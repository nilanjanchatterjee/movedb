# Global Species Telemetry Research Database
     
     
This is a database that compiles records of published studies using Global positioning based telemetry on wild animals. Each point represents a study location associated with metadata including species name, journal, year, and DOI. Initial searches were carried out in August 2025. 

    More than 75,000 records were found with our search strings. The records were screened using both manual and machine learning approaches. Please go through the methods page in case you want to know more about the approaches. In the current version, the data consist only records of mammals but in future, we want to expand it to birds and other taxa.  
    
    Data were harmonized and standardized using reproducible workflows in R, and exported as GeoJSON for web visualization. For further queries, please reach out to xxx@xyz.com
